This patch is added on top of release v2.3.4-rc0820 with following updates - 
1. 3DS2 support for Google Pay
2. 3DS2 support for Instant purchase

Worldpay Magento2 Plugin - 

This version is compatible with Magento 2.3.4 with following key features:
1. Credit card payment
2. Hosted payment page solution
3. Alternative payment methods like Paypal, AliPay, Klarna etc.
4. Mobile wallets like Google Pay and Apple pay 
5. Client side encryption for direct integration
6. 3DS Authentication
7. Exemption Engine
8. Latin America Specific Payment 
9. Mobile order and Telephone Order support for customer support executive
10. Shopper tokens and Merchant tokens
11. Multi store and multi merchant support
